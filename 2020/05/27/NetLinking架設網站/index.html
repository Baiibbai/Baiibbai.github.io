NetLinking架設網站

文件下載

  • Clone NetLinking網站資料夾 XSG.ME.NetLink

執行環境設定

  • Visual Studio 2019安裝設定

  • 下載 Visual Studio 2019

  • 社群版

  • 下載工作負載

  • 下載個別元件

IIS權限設定

  • 控制台(設定)->應用程式

  • 應用程式->程式和功能

  • 程式和功能->開啟或關閉Windows功能

  • 開啟或關閉Windows功能



  • 如未勾選完全，執行會發生錯誤

    黑方框 : 部分勾選
    打勾 : 全部內容勾選

MONGODB

下載 MongoDB

建立執行檔

  • 開啟clone下來的資料夾

  • XSG.ME.Netlink.sln

  • 建置

    建置方案
    發佈XSG.ME.NetLink

  • 建立發佈檔 -> 新增

  • 選擇資料夾

  • 將檔案放在D槽，自行新增的資料夾

  • 必須將資料夾放在D槽，否則會出現錯誤

IIS基本設定

  • 開啟IIS管理員 ->點擊站台 ->開啟基本設定

  • 實體路徑選擇剛剛建立好的發佈檔

  • 啟動或重新啟動 : 同步建立檔

  • 即可啟動 開啟網頁測試

  • Ex. http://(本機位置)/api/GETEvents


    出現此頁面表示發佈成功，即可開始使用網站
